# UniSERPRO

en: A [Moodle](https://moodle.org) theme developed to serve as main theme for [UniSERPRO](https://moodle.ead.serpro.gov.br/), [SERPRO's](http://serpro.gov.br) corporate university. It is soon to be put on-line.
